---
description: How to run the anime draft website locally
---

1. Open a terminal in the project root (`c:\Users\kaira\anime-draft`).
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Wait for the output to say "Ready in ..." or showing the URL.
4. Open your web browser and navigate to:
   [http://localhost:3000](http://localhost:3000)
